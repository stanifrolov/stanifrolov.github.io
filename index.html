Research

I'm interested in image synthesis, and particularly, controlled image synthesis. There are multiple ways to gain control over the image generation process such as conditioning or learning disentangled representations. So far, I've worked on text-to-image, and layout-to-image models. In contrast to class labels, text offers a flexible and intuitive interface, while layouts consisting of bounding boxes or masks + labels allows fine-grained positional control. I'm also interested in automatic evaluation techniques for vision & language systems which is a challenging problem.

If you are interested in similar topics, want to chat or collaborate with me, feel free to contact me.

Publications
Leveraging Visual Question Answering to Improve Text-to-Image Synthesis
Stanislav Frolov, Shailza Jolly, Jörn Hees, Andreas Dengel
LANTERN workshop at COLING, 2020

We concatenate question answer pairs from the VQA dataset and use them as additional training samples in a text-to-image synthesis model. A standard VQA model can then be used to get an additional VQA loss. This improves both image quality, and image-text alignment.
